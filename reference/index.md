# Package index

## GeoComplexity Computation

- [`geocd_vector()`](https://ausgis.github.io/geocomplexity/reference/geocd_vector.md)
  : geocomplexity for spatial vector data based on spatial dependence
- [`geocs_vector()`](https://ausgis.github.io/geocomplexity/reference/geocs_vector.md)
  : geocomplexity for spatial vector data based on geographical
  similarity
- [`geocd_raster()`](https://ausgis.github.io/geocomplexity/reference/geocd_raster.md)
  : geocomplexity for spatial raster data based on spatial dependence
- [`geocs_raster()`](https://ausgis.github.io/geocomplexity/reference/geocs_raster.md)
  : geocomplexity for spatial raster data based on geographical
  similarity

## GeoComplexity Application

- [`geocs_swm()`](https://ausgis.github.io/geocomplexity/reference/geocs_swm.md)
  : constructing spatial weight matrix based on geocomplexity with
  similar geographical configurations
- [`geocd_swm()`](https://ausgis.github.io/geocomplexity/reference/geocd_swm.md)
  : constructing spatial weight matrix based on geocomplexity with
  spatial dependence
- [`gwr_geoc()`](https://ausgis.github.io/geocomplexity/reference/gwr_geoc.md)
  : geographical complexity-geographically weighted regression
